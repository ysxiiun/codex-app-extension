- 本项目只做 Codex App 本地运行时增强，不修改 Codex App 安装包体、账号数据或历史会话数据。
- 用户可配置能力、新增 CLI 参数、新增环境变量、新增诊断字段必须同步更新 README。
- 旧兼容入口默认保留，除非用户明确要求清理。
